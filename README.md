🧠 AI-Based Anomaly Detection Platform
🚀 Overview
In today’s fast-paced healthcare environment, continuous patient monitoring is important but not always efficient with traditional methods.
This project introduces a smart system that tracks patient vital signs in real time and identifies unusual patterns instantly.

⚡ How It Works
The system continuously processes live physiological data using machine learning models.
Instead of relying on fixed limits, it learns patterns from data and detects anomalies dynamically as they occur.

📊 Detection & Response
When an abnormal condition is detected:

The event is recorded in a PostgreSQL database
It is immediately reflected on a Flask-based dashboard
Alerts are triggered to ensure quick attention and response
🎯 Why This Project
The aim of this project is to move beyond basic monitoring systems and build a more intelligent solution that can assist in early detection and better decision-making.

💡 Final Note
This project combines AI, real-time processing, and visualization to create a practical healthcare solution that can make monitoring more efficient and reliable.
